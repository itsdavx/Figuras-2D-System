# Aplicación de Figuras Geométricas 2D en C#

## Descripción

Aplicación de escritorio desarrollada en C# con Windows Forms (.NET Framework) que permite dibujar figuras geométricas básicas en 2D utilizando la clase `Graphics`. El sistema incluye un menú interactivo para seleccionar distintas figuras, las cuales se renderizan correctamente posicionadas, con colores diferenciados y escalado proporcional.

---

## Objetivos

### Objetivo General
Desarrollar una aplicación gráfica que permita representar figuras geométricas en 2D aplicando conceptos de programación orientada a objetos y manipulación de gráficos.

### Objetivos Específicos
- Implementar el uso de la clase `Graphics` para renderizado.
- Aplicar clases como `Pen`, `Brush` y `Point`.
- Diseñar una interfaz clara y funcional.
- Garantizar el escalado proporcional de las figuras.

---

## Funcionalidades

- Menú para selección de figuras geométricas.
- Dibujo de figuras en un área gráfica.
- Posicionamiento adecuado en pantalla.
- Uso de colores personalizados.
- Escalado proporcional de las figuras.
